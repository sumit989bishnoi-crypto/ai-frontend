🧠 CodeRescue

CodeRescue is an AI-powered tool that analyzes and fixes broken code automatically, helping developers save time and learn from their mistakes.

🚀 Live Demo

👉 https://sumit989bishnoi-crypto.github.io/ai-frontend/

🛠️ How to Use
Open the live demo link above
Paste your broken code into the input box
View detected errors and explanations
Copy the corrected code and use it in your project
⚙️ Tech Stack
HTML, JavaScript (Frontend)
Python (Flask Backend)
Hugging Face Spaces (Deployment & AI backend)
✨ Features
AI-powered code debugging
Automatic error detection and fixing
No login required
Fast and efficient responses
Automatic retry on failed requests
Beginner-friendly interface
💡 Why Use CodeRescue?

CodeRescue not only fixes your code but also helps you understand the mistakes, making it a useful tool for both beginners and experienced developers.
